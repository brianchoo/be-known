# Freelance Website

A simple freelance website that is built using Next.js

## What technology is used

1. Next.js
2. Tailwind
3. Nodemailer

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install dependencies.

```bash
npm install
```

## How to run locally

```python
npm run dev

```

## How to build for production

```python
npm run build

```
